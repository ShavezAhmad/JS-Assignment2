Q7) Write a function that accepts parameter n and returns factorial of n
var n=5;
console.log(`Factorial of ${n} is ${fact(n)}`);
fact(n)
function fact(n)
{
    let f=1;
    for(let i=1;i<=n;i++){
        f=f*i;
    }
    return f;
}


O/P -> Factorial of 5 is 120
